# Руководство пользователя
## CoffeeCompanyKiosk

### Для клиента

1. Запустите приложение.
2. На вкладке «Меню» выберите категорию блюд.
3. Нажмите кнопку «+» у нужного блюда, чтобы добавить его в корзину.
4. Перейдите на вкладку «Корзина» для просмотра выбранных позиций.
5. Нажмите «Оформить заказ», введите телефон и выберите способ получения.
6. Получите чек на экране и сообщите номер заказа сотруднику.

### Для администратора

1. Нажмите кнопку «Вход (админ)» на главном экране.
2. Введите логин `admin` и пароль `admin123`.
3. В открывшемся окне просмотрите все заказы.
4. Нажмите «Экспорт в CSV» для выгрузки данных.
5. Для обновления меню отредактируйте файл `menu.json` и перезапустите приложение.
