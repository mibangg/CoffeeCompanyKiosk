# CoffeeCompanyKiosk

Приложение-киоск для самостоятельного оформления заказов в кофейне «Coffee Company».

## Основные функции

- Просмотр меню с категориями и блюдами
- Поиск блюд по названию
- Добавление блюд в корзину, изменение количества, удаление позиций
- Оформление заказа (ввод телефона, выбор самовывоза или доставки)
- Формирование чека и сохранение в текстовый файл
- Панель администратора (просмотр заказов, экспорт в CSV)

## Технологии

- Python 3.10+
- Tkinter (графический интерфейс)
- JSON (хранение данных)

## Использование

- **Клиент:** выбирайте блюда, добавляйте в корзину, оформляйте заказ.
- **Администратор:** нажмите кнопку «Вход (админ)», введите логин `admin` и пароль `admin123`.
